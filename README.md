# chunky-mcp
An MCP server to handle chunking and reading large responses

## Project Setup
### Using UV
1. Install `uv`
2. `uv pip install git+https://github.com/actual-username/chunky-mcp.git`

### Cloning 
1. `git clone https://github.com/actual-username/chunky-mcp.git`
2. `cd chunky-mcp`
3. `uv pip install -e .`
